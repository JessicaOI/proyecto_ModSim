1. Descripcin del Problema (Investigacin)

    Objetivo: Desarrollar un modelo que simule la propagacin de una enfermedad infecciosa en una poblacin determinada. Justificacin: Entender cmo se propaga una enfermedad puede ayudar a tomar decisiones informadas sobre medidas de salud pblica, como vacunacin, cuarentenas, o restricciones de movilidad.

  2. Tcnica Utilizada (Aplicacin)

    Modelo Epidemiolgico: Usaremos el modelo SIR, que divide la poblacin en tres categoras: Susceptible (S), Infectado (I), y Recuperado (R). Parmetros Clave: Tasa de transmisin, tasa de recuperacin, y poblacin inicial en cada categora.

  3. Implementacin (Solucin)

    Construccin del Modelo: Crearemos un modelo matemtico basado en ecuaciones diferenciales que describan la transicin entre los estados S, I, y R. Simulacin: Realizaremos simulaciones variando los parmetros clave para observar diferentes escenarios de propagacin.

  4. Calibracin y Validacin

    Calibracin: Ajustaremos los parmetros del modelo utilizando datos reales de un brote de enfermedad especfico. Validacin: Compararemos las predicciones del modelo con los datos reales para evaluar su precisin.

  5. Anlisis de Sensibilidad

    Objetivo: Determinar cmo la variacin en los parmetros afecta los resultados del modelo. Mtodo: Variaremos sistemticamente los parmetros (como la tasa de transmisin) y observaremos los cambios en la dinmica de la epidemia.

  6. Recomendaciones (Reflexin)

    Aplicaciones del Modelo: Ofreceremos recomendaciones sobre cmo utilizar el modelo para apoyar la toma de decisiones en salud pblica. Limitaciones y Mejoras Futuras: Discutiremos las limitaciones del modelo y posibles extensiones o mejoras.

import numpy as np
import matplotlib.pyplot as plt
from scipy.integrate import odeint
# Implementacin del modelo SEIR

def modelo_seir_con_nacimientos_muertes(y, t, N, beta, sigma, gamma, mu):
    S, E, I, R = y

    beta_actual = beta

    # Ecuaciones del modelo SEIR
    dS_dt = mu * N - beta_actual * S * I / N - mu * S
    dE_dt = beta_actual * S * I / N - sigma * E - mu * E
    dI_dt = sigma * E - gamma * I - mu * I
    dR_dt = gamma * I - mu * R
    return [dS_dt, dE_dt, dI_dt, dR_dt]

# Parmetros ajustados para COVID-19
N = 1      # Poblacin total (normalizada a 1)
beta_covid = 2.5 / 10  # Estimacin: R0 de 2.5 y un periodo infeccioso de 10 das
sigma_covid = 1/5.0    # Periodo de incubacin de 5 das
gamma_covid = 1/10.0   # Duracin promedio de la infeccin de 10 das
t = np.linspace(0, 160, 160) # Tiempo de simulacin (en das)
mu_covid = 1/(70*365.0)  # Tasa de nacimiento/muerte

# Condiciones iniciales
S0 = 0.99  # Porcentaje de la poblacin susceptible al inicio
E0 = 0.01  # Porcentaje de la poblacin expuesta al inicio
I0 = 0.0   # Porcentaje de la poblacin infectada al inicio
R0 = 0.0   # Porcentaje de la poblacin recuperada al inicio
condiciones_iniciales_seir = [S0, E0, I0, R0]

# Resolucin del sistema de ecuaciones para el modelo SEIR con parmetros de COVID-19
solucion_covid = odeint(modelo_seir_con_nacimientos_muertes, condiciones_iniciales_seir, t, args=(N, beta_covid, sigma_covid, gamma_covid, mu_covid))

# Extraccin de resultados para el modelo SEIR con parmetros de COVID-19
S_covid, E_covid, I_covid, R_covid = solucion_covid.T

# Visualizacin de resultados para el modelo SEIR con parmetros de COVID-19
plt.figure(figsize=(10,6))
plt.plot(t, S_covid, label='Susceptibles')
plt.plot(t, E_covid, label='Expuestos')
plt.plot(t, I_covid, label='Infectados')
plt.plot(t, R_covid, label='Recuperados')
plt.xlabel('Das')
plt.ylabel('Proporcin de la poblacin')
plt.title('Simulacin de una Epidemia de COVID-19 con el Modelo SEIR (con Nacimientos y Muertes)')
plt.legend()
plt.grid(True)
plt.show()

import numpy as np
import matplotlib.pyplot as plt
from scipy.integrate import odeint
import math

# Implementacin del modelo SEIR con un evento de confinamiento temporal
def modelo_seir_confinamiento(y, t, N, beta, beta_reducido, sigma, gamma, mu, inicio_confinamiento, fin_confinamiento):
    S, E, I, R = y

    # Aplicar una reduccin en la tasa de transmisin durante el perodo de confinamiento
    if inicio_confinamiento <= t <= fin_confinamiento:
        beta_actual = beta_reducido
    else:
        beta_actual = beta

    # Ecuaciones del modelo SEIR
    dS_dt = mu * N - beta_actual * S * I / N - mu * S
    dE_dt = beta_actual * S * I / N - sigma * E - mu * E
    dI_dt = sigma * E - gamma * I - mu * I
    dR_dt = gamma * I - mu * R
    return [dS_dt, dE_dt, dI_dt, dR_dt]

# Parmetros del modelo con confinamiento
beta = 0.3  # Tasa de transmisin normal
beta_reducido = 0.1  # Tasa de transmisin reducida durante el confinamiento
inicio_confinamiento = 30  # Da de inicio del confinamiento
fin_confinamiento = 90   # Da de fin del confinamiento

# Resolucin del sistema de ecuaciones con confinamiento
solucion_seir_confinamiento = odeint(modelo_seir_confinamiento, condiciones_iniciales_seir, t, args=(N, beta, beta_reducido, sigma, gamma, mu, inicio_confinamiento, fin_confinamiento))

# Extraccin de resultados con confinamiento
S_seir_confinamiento, E_seir_confinamiento, I_seir_confinamiento, R_seir_confinamiento = solucion_seir_confinamiento.T

# Visualizacin de resultados con confinamiento
plt.figure(figsize=(10,6))
plt.plot(t, S_seir_confinamiento, label='Susceptibles')
plt.plot(t, E_seir_confinamiento, label='Expuestos')

plt.axvline(x=inicio_confinamiento, color='grey', linestyle='--', lw=2, label='Inicio del Confinamiento')

plt.xlabel('Das')
plt.ylabel('Proporcin de la poblacin')
plt.title('Modelo SEIR con Evento de Confinamiento Temporal')
